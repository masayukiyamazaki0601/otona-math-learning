Chapter 04 / Unit 46

式の展開と基本公式

Step 01

かっこを外すだけなのに、なぜ間違えやすいの？

「展開」は、文字式の計算で最初に学ぶ重要事項の一つです。

一見すると、かっこを外すだけの単純な計算のように思えるかもしれません。しかし、符号の処理を間違えたり、特定の項への掛け算を忘れてしまったりすることで、思わぬミスにつながることが非常に多い箇所です。

展開の正しい考え方を整理しておくと、複雑な数式もスムーズに扱えるようになります。また、この後に学ぶ「因数分解」や「二次方程式」の基礎体力となるため、ここで確実な計算力を身に付けましょう。

Step 02

展開とは

数学における展開（てんかい）とは、かっこを含んだ積の形をした式を、かっこを外して単項式の和（簡単な形）に表すことです。

具体的な計算例

例えば、$3(x+2)$ という式を展開する場合、外側にある $3$ をかっこの中にあるすべての項（ $x$ と $2$ ）に掛け合わせます。
 $3(x+2) = 3 \times x + 3 \times 2$
     $= \mathbf{3x + 6}$
「かっこの前の数や文字は、中のすべての項に漏れなく掛ける」というのが展開の基本大原則です。

Step 03

分配法則を使う

展開では、分配法則（ぶんぱいほうそく）という数式の基本的な性質を利用します。

分配法則とは、かっこの前（または後ろ）にある数や文字を、中の各項に分配して掛けるルールです。

【体験】分配法則シミュレーター（長方形の面積）
左面積 ab = 20.00 ㎠
右面積 ac = 15.00 ㎠
合計 a(b+c) = 35.00 ㎠
Step 04

文字どうしの展開

数と文字だけでなく、文字どうしを掛け合わせる場合も、全く同一のルール（分配法則）が適用されます。

文字展開の面積図解： $x(x+4) = x^2 + 4x$

式 $x(x+4)$ の展開は、縦が $x$、横が $x+4$ の長方形の面積と捉えられます。
 $x(x+4) = x \times x + x \times 4$
      $= \mathbf{x^2 + 4x}$
もう一つの例として、係数がつく場合も同様です：
 $2a(3a-5) = \mathbf{6a^2 - 10a}$

Step 05

基本公式（乗法公式）をマスターする

式の展開には、頻繁に出現するパターン化された計算が存在します。よく使う以下の3つの**基本公式**をマスターしておくと、圧倒的に素早く正確に計算できるようになります。

① 平方の公式（和の2乗）シミュレーター
$(a+b)^2 = a^2 + 2ab + b^2$
正方形 a² = 25.00 ㎠
長方形 2ab = 25.00 ㎠
正方形 b² = 6.25 ㎠
合計 (a+b)² = 56.25 ㎠
② 平方の公式（差の2乗）
$(a-b)^2 = a^2 - 2ab + b^2$

全体 $a^2$ から、右の帯 $ab$ と下の帯 $ab$ を引くと、右下の重なる正方形 $b^2$ が2回重複して引かれてしまいます。そのため、最後に $b^2$ を1つ足し戻すことで帳尻を合わせます。

【計算例】
 $(x-5)^2 = x^2 - 2\times(x\times5) + 5^2$
      $= \mathbf{x^2 - 10x + 25}$

③ 和と差の積の公式
$(a+b)(a-b) = a^2 - b^2$

1辺が $a$ の正方形から、右下の $b^2$ を切り取ったL字型の面積（左図）は、はみ出た長方形（縦 $b$、横 $a-b$）を右上に回転させて移動させることで、縦が $a-b$、横が $a+b$ の1つの大きな長方形（右図）にぴったり組み替えることができます。

【計算例】
 $(x+4)(x-4) = x^2 - 4^2$
        $= \mathbf{x^2 - 16}$

Step 06

展開するときのポイントと注意点

展開計算において、最も発生しやすい計算ミスは**「掛け忘れ」**と**「符号ミス」**です。

よくある間違いの例

間違い①：分配の掛け忘れ
 $2(x+5) \rightarrow 2x + 5$ （うしろの $5$ に掛け忘れる）
 正しくは： $2x + 10$

間違い②：平方公式の真ん中の項の消失
 $(a-b)^2 \rightarrow a^2 - b^2$ （真ん中の $-2ab$ が消えてしまう）
 正しくは： $a^2 - 2ab + b^2$

公式を単にアルファベットの羅列として丸暗記するのではなく、前述した「面積図」のビジュアルイメージと結びつけて記憶することで、計算ミスを大幅に減らすことができます。

Step 07

まとめ

  • 展開とは、かっこを含んだ積の形をした式を、かっこを外して単項式の和に表すこと。
  • 分配法則 $a(b+c) = ab + ac$ を基本とし、かっこ内のすべての項に掛け合わせる。
  • 特によく使われる以下の3公式は、計算スピード向上のために暗記する。
  • ① $(a+b)^2 = a^2 + 2ab + b^2$
    ② $(a-b)^2 = a^2 - 2ab + b^2$
    ③ $(a+b)(a-b) = a^2 - b^2$
番外編 (Column)

【番外編】ビジネスでの活用：『単価値上げ × 客数減少』の売上予測を数式展開で要素分解する

ビジネスにおいて、商品の値上げや値下げを行う際、「売上全体がどう変化するか」を予測することは極めて重要です。この予測シミュレーションの現場でも、式の展開公式が「要因分析のツール」として活用されています。

【体験】単価値上げ・客数減少の売上シミュレーター
売上変動の要因分解:
① 単価値上げ効果 (200x) = +10,000 円
② 客数減少効果 (400y) = -12,000 円
③ 相乗効果 (xy) = -1,500 円
新しい売上高: 76,500 円
(元の売上: 80,000 円)

このシミュレーションの計算式は、まさに多項式の掛け算 $(400 + x)(200 + y)$ です。これを展開することで、単に売上の総額がわかるだけでなく、**「値上げ効果によるプラス（ $200x$ ）」**と**「客数減少によるマイナス（ $400y$ ）」**がそれぞれどれだけ発生しているかを個別に割り出す（要素分解する）ことが可能になります。数式の展開は、ビジネスにおける収益の要因分析を行うための強力な武器なのです。
