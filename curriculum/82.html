Chapter 06 / Unit 82

逆・裏・対偶の関係性

Step 01

命題の3つの変形パターン

元の命題「$p$ ならば $q$」に対して、言葉の順番を入れ替えたり、否定（〜でない）をつけたりすることで、全く新しい3つの命題を作ることができます。
これを「逆（ぎゃく）」「裏（うら）」「対偶（たいぐう）」と呼びます。

  • 元の命題：$p \implies q$（$p$ならば$q$）
  • 逆（ぎゃく）：$q \implies p$（前後を入れ替える）
  • 裏（うら）：$\text{not } p \implies \text{not } q$（前後の位置はそのままで、両方否定する）
  • 対偶（たいぐう）：$\text{not } q \implies \text{not } p$（前後を入れ替えて、さらに両方否定する）
Step 02

最強の法則「対偶の真偽は、元の命題と必ず一致する」

この3つの変形の中で、最も重要でビジネスでも使えるのが「対偶（たいぐう）」です。
論理学には、「元の命題が正しい（真）ならば、その対偶も100%絶対に正しい（真）になる」という絶対法則があります。

逆に言えば、「逆」と「裏」は、元の命題が正しくても間違っている（偽）になることがよくあります。
多くの人が議論の中で、この「逆」や「裏」を正しいと思い込んでしまうことで、論理の飛躍や誤解が生まれます。

Step 03

元の命題「犬ならば哺乳類である（真）」を基準とします。

番外編 (Column)

【番外編】クレーム対応を乗り切る「対偶」の論理

対偶の「元の命題と真偽が一致する」という性質は、ビジネスの交渉やクレーム対応において、相手の隠れた意図（矛盾）を暴くのに役立ちます。

「誠意を見せろ！」の矛盾を突く

悪質なクレーマーが「金銭を要求しているわけではないが、誠意を見せろ」と言ってきたとします。もしこの要求の本質が「誠意を見せる＝お金を払う」という命題（暗黙の要求）であるならば、その対偶をとってみましょう。

対偶は「お金を払わない ならば 誠意を見せたことにならない」となります。
相手に「つまり、金銭を支払わなければ誠意を見せたことにはならない、という意味でしょうか？」と対偶を用いて確認することで、「金銭を要求しているわけではない」という建前と矛盾していることを論理的に浮き彫りにし、相手の不当な要求（恐喝）を牽制することができます。

Step 05

まとめ

  • 元の命題 $p \implies q$ に対し、言葉の入れ替えと否定の組み合わせで「逆・裏・対偶」が作れる。
  • 逆（$q \implies p$）と裏（$\text{not } p \implies \text{not } q$）は、元の命題が真であっても、真になるとは限らない（偽になることが多い）。
  • 対偶（$\text{not } q \implies \text{not } p$）は、元の命題と必ず真偽が一致する。
  • 一見わかりにくい命題に出会った時は、「対偶」に変換してみると本質が見えやすくなる。
