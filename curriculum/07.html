Chapter 02 / Unit 07

四則演算の基本法則（加法・減法）

Step 01

加法（足し算）の意味

足し算（加法）は、2つの数量を「合わせる」操作です。もっとも基本的な演算であり、すべての計算の土台になります。

加法の基本

$a + b = c$

  • $a$ に $b$ を「加える」と $c$ になる
  • 例: $3 + 5 = 8$（3個のりんごに5個のりんごを加えると、全部で8個）

足し算には、次の3つの重要な性質があります。

加法の3つの性質

  1. 交換法則: $a + b = b + a$（順番を変えても答えは同じ）
  2. 結合法則: $(a + b) + c = a + (b + c)$（どこから足しても答えは同じ）
  3. 単位元の存在: $a + 0 = 0 + a = a$（0を足しても変わらない）
Step 02

減法（引き算）の意味

引き算（減法）は、数量から一部を「取り除く」操作です。足し算の逆の操作と考えることができます。

減法の基本

$a - b = c$

  • $a$ から $b$ を「取り去る」と $c$ になる
  • 例: $8 - 5 = 3$（8個のりんごから5個を取り除くと、3個残る）

引き算の注意点

  • 交換法則は 成り立たない: $a - b \neq b - a$（ただし $a = b$ の場合を除く）
  • $a - b = c$ のとき、$c + b = a$ が成り立つ（足し算で検算できる）
Step 03

加法と減法の関係

足し算と引き算は、互いに逆の関係にあります。この関係を理解すると、計算の検算や暗算がぐっと楽になります。

加法と減法の相互関係

  • $a + b = c$ ならば $c - b = a$ かつ $c - a = b$
  • $a - b = c$ ならば $c + b = a$

たとえば $25 + 17 = 42$ が正しいかどうかは、$42 - 17 = 25$ で確認できます。買い物でお釣りを計算するときも、この関係を使っています。

Step 04

ビジネスでの活用例

足し算と引き算は、ビジネスのさまざまな場面で使われています。

例：売上の計算

ある店舗の1日の取引:

  • 朝の売上: $45,000$円
  • 昼の売上: $62,000$円
  • 夜の売上: $38,000$円
  • 合計: $45,000 + 62,000 + 38,000 = 145,000$円

経費 $98,000$円を差し引いた利益:

  • $145,000 - 98,000 = 47,000$円

このように、加法と減法は「増減」を把握するための基本的な道具です。複雑なビジネス計算も、このシンプルな足し算と引き算の組み合わせでできています。

Step 05

まとめ

  • 加法（足し算）は数量を「合わせる」操作。
  • 加法には交換法則・結合法則が成り立つ。
  • 減法（引き算）は数量から「取り去る」操作。
  • 加法と減法は互いに逆の関係にあり、検算に使える。
