Chapter 03 / Unit 31

比の性質と比例式の解法

Step 01

なぜ比を学ぶのか

私たちの日常やビジネスには、絶対的な数値だけでなく、**「数量同士の割合（バランス）」**を捉えるべき場面が多く存在します。例えば、画面の「アスペクト比（16：9）」や、カクテルのレシピ、デザインの「黄金比」などです。

相対的な「関係性」を固定する

比を用いると、全体の規模が変化しても、要素間のバランス（関係性）を一定に保つことができます。そして、この比のバランスを利用して「片方が2倍になったなら、もう片方はどうなるか」という未知の数値を逆算する手法が比例式です。

Step 02

比・比の性質と「最も簡単な比」

数学の用語解説

・比（ひ）：2つの数量の割合の関係を表したもの。 $a : b$ と書く。
・最も簡単な比：比の両方の数を公約数で割り、それ以上小さくできない整数の比。

比の最も重要な性質は、**「両方の数に同じ数を掛けたり、同じ数で割ったりしても、比の価値は変わらない」**ということです。

比の縮小操作（約分のイメージ）
・掛け算： $2 : 3$ の両辺を $2$ 倍 → $4 : 6$
・割り算： $12 : 18$ の両辺を最大公約数の $6$ で割る → $2 : 3$ （最も簡単な比）
Step 03

比例式と「内外の積」の黄金律

2つの比が等しいことをイコール「$=$」で結んだ式を**比例式（ひれいしき）**と呼びます。比例式を解くための、最も強力な黄金ルールがこちらです。

外側どうしの積 ＝ 内側どうしの積
$$a : b = c : d \implies ad = bc$$

この性質を使うことで、比の式「 $:$ 」を、見慣れた一次方程式「掛け算の等式」へと一瞬で変換することができます。

Step 04

比例式の具体的な解き方

比例式の性質を利用して、未知数 $x$ を含む比例式を解く手順を解説します。

例題： $3 : 5 = x : 20$ を解きなさい。
1. 内外の積の公式を適用する：
$\quad 3 \times 20 = 5 \times x$

2. 式を整理して一次方程式にする（見やすく左右を入れ替える）：
$\quad 5x = 60$

3. 方程式を解く（両辺を $5$ で割る）：
$\quad x = 12$

4. 答えを検証する：
$\quad$ 元の式の右辺に $x = 12$ を代入すると $12 : 20$ となる。両辺を $4$ で割って最も簡単な比にすると $3 : 5$ となり、左辺と完全に一致する。よって 答え： $x = 12$。
Step 05

まとめ

  • 比（ $a : b$ ）は、全体の規模に左右されない**「相対的なバランス」**を表す。
  • 比の両辺に**同じ数を掛けたり割ったりしても、比の関係は変化しない**。
  • 比例式（ $a : b = c : d$ ）は、**「外側の積＝内側の積（ $ad = bc$ ）」**を利用して一次方程式に直して解く。
番外編 (Column)

【番外編】身近なAIとの繋がり：スマホカメラの「自動ズーム」と美肌フィルターの追従

スマホのカメラでポートレート（背景ぼかし）撮影や美肌フィルターを使うとき、ズーム（拡大・縮小）をしてもフィルターの位置がズレずに、顔にぴったり張り付いたまま動きます。ここでも「比例式」の計算が活躍しています。

ズーム倍率に合わせてフィルターの大きさを自動スケーリングする

スマホ内の画像処理AIは、元の画面サイズでの「顔の大きさ（幅 $150\text{px}$）」と「画面全体の幅（$1920\text{px}$）」の比を計算しています。

ズームして画面全体の表示が小さくなった（例えば幅 $640\text{px}$ にトリミングした）場合、新しい画面での「顔の大きさ $x$」を、以下の比例式で計算して求めます：

$$x : 150 = 640 : 1920$$ （※内側の積＝外側の積を使って解くと、 $x = 50\text{ px}$ ）

この計算により、AIは拡大された顔に合わせて、美肌フィルターのサイズを自動で $50\text{px}$ 分に縮小し、違和感のないように重ね合わせます。スマホのカメラが高精度に私たちの顔や被写体を追いかけて加工できるのは、この比率の計算が毎秒数十回も一瞬で実行されているからなのです。
