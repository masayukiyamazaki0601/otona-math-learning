Chapter 02 / Unit 09

計算の優先順位と括弧の扱い

Step 01

なぜ計算の順序にルールがあるのか？

例えば、$2 + 3 \times 4$ という式を見てみましょう。

この式を左から順番にそのまま計算すると、$2 + 3 = 5$、さらに $5 \times 4 = 20$ となりそうですが、これは数学上の誤りです。正しい答えは、$3 \times 4 = 12$ を先に計算し、それに 2 を足した 14 になります。

なぜ掛け算が足し算よりも優先されるのでしょうか。これは掛け算の定義が「同じ数を繰り返し足す操作をまとめたもの（＝累加の塊）」だからです。つまり、$3 \times 4$ は「$3$ が $4$ つある塊（ひとまとまりの数量）」を示しており、個別の数値 $2$ とは結びつきが強いため、バラバラにしてはいけません。このように、計算には世界共通の「優先順位（Precedence）」が存在します。

【計算優先順位の絶対ルール】

  1. 括弧（かっこ）の中が最優先。
  2. 掛け算・割り算（乗除）が次に優先。
  3. 足し算・引き算（加減）は最後に計算する。
  4. 同じ優先順位の計算が並ぶ場合は、原則として左から右へ順に計算する。

Step 02

括弧の力と、複数括弧の順序関係

括弧（かっこ）は、数理上の標準の優先順位を強制的に書き換え、特定の計算を最優先に行わせるための「指揮官」の役割を果たします。

括弧を使うか使わないかで、計算順序と結果がどう変わるかを比較します。

  • $10 - 4 + 2 = 6 + 2 = 8$
    括弧なしの場合、加減は優先度が同じなので左から順に計算します。
  • $10 - (4 + 2) = 10 - 6 = 4$
    括弧があるため、後ろの $(4+2)$ を先に計算し、引き算を最後に行います。答えが変わります。

さらに、式の中に括弧が何重にも入り組んでいる（入れ子構造）場合は、必ず「一番内側の括弧」から順番に外側に向けて計算を解いていきます。

数学における括弧の種類と計算順序は以下の通りです。

丸括弧 ( )  →  角括弧 [ ]  →  波括弧 { }

※常に最も内側の「丸括弧 ( ) 」から最優先で展開します。

数理パズル：テンパズル (Make 10)

提示された4つの数字を「必ず1回ずつ」すべて使い、加減乗除（$+, -, \times, \div$）と括弧を組み合わせて、計算結果がちょうど 10 になる式を完成させてください。

Step 03

ビジネスで頻発するExcelの括弧ミスと財務リスク

仕事でExcelやプログラミングを扱う際、この優先順位のルールを誤って認識していると、重大なシステムエラーや誤った財務報告書（見積書など）を提出する原因になります。最もありがちな事例は「利益率（粗利率）の計算」です。

事例: 売上高100万円、仕入原価60万円から粗利率（40%）を求める式

計算方法 Excel入力式 Excelの解釈と計算順序 結果 判定
誤った式 = 100 - 60 / 100 先に乗除を行うため、先に $60 \div 100 = 0.6$ が計算され、その後 $100 - 0.6$ が実行される。 99.4% NG (大損害)
正しい式 = (100 - 60) / 100 括弧をつけたことで先に引き算（利益額: 40万円）が計算され、それを売上高で割る。 40% OK (正確)

たった1組の括弧を書き忘れるだけで、粗利率「40%」が「99.4%」という全く異なる数字として計算されてしまいます。ビジネスにおける数理処理では、常に乗除が優先されることを念頭に置き、必要な括弧を漏らさず記述することが極めて重要です。

Step 05

本単元のまとめ

  • 計算の優先ルール: 「括弧の中」→「乗除（掛け算・割り算）」→「加減（足し算・引き算）」の順序を厳守します。同じ優先順位なら左から右へ計算します。
  • 括弧の優先効果: 括弧は、標準ルールをオーバーライドして優先度を最上位に引き上げます。
  • 何重もの括弧: 複雑な数式では、最も内側の括弧（丸括弧）から順に外側へ向けて解いていきます。
  • ビジネスにおける重要性: Excelでの計算式入力時など、括弧の付け忘れは桁違いの数値算出エラーを招く重大なリスクとなります。
