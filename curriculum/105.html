Chapter 08-05

次元削減（PCA）と固有値・固有ベクトルの幾何学的意味

多次元の複雑なデータを、情報（ばらつき）を失わずに低次元へ要約する「主成分分析（PCA）」。その幾何学的な本質と、共分散行列の固有値問題へと帰着する数理の全貌を紐解きます。

Step 01

多次元データと「次元削減」の必要性

現実のビジネスや科学研究で扱うデータは、非常に多くの変数（次元数）を持っています。 顧客分析を例に取ると、購入金額、購入頻度、Webサイトの滞在時間、年齢、居住地域など、数十から数百の項目が同時に記録されます。 このように次元数が多いデータをそのまま扱うとき、データ分析は2つの大きな困難に直面します。

  • データの可視化が困難になる点：人間が視覚的に認識できる空間は3次元までです。それ以上の高次元データを直接グラフに描いてデータ構造を理解することは不可能です。
  • 「次元の呪い」の発生：次元数が増えるにつれて、データを表現する空間の体積が指数関数的に増加します。その結果、データ点が空間内でスカスカにばらけて存在し、予測モデルの精度が低下する現象を「次元の呪い」と呼びます。

これらの困難を解決する手法が「次元削減」です。 次元削減は、データの元々持つ本質的な情報（ばらつき）をできる限り残しながら、変数の数をギュッと絞り込む作業を指します。 その代表的な手法が、主成分分析（PCA: Principal Component Analysis）です。

Step 02

「情報の損失を最小化する」とはどういうことか

2次元平面上のデータを、直線の1次元空間へ次元削減する場面を考えます。 高次元のデータを低次元に射影（投影）するとき、どのような方向に射影すれば、データの情報を最も失わずに済むでしょうか。 統計学において、データの「情報」とは「データのばらつき（分散）」を意味します。 データが広く散らばっている方向ほど、データごとの違いを識別するための重要な情報が多く含まれていると考えられます。

したがって、情報の損失を最小限にするための射影方向は、「射影後のデータの分散が最大になる方向」です。 射影後のデータが最も大きく散らばる直線の向きを探し出すことが、PCAにおける最適化の目的となります。

Step 03

射影されたデータの分散を表す数式モデル

この問題を数式として構築します。 まず、手元にあるデータ点を $x_i = [x_{i1}, x_{i2}, \dots, x_{id}]^T$ とし、あらかじめ中心化（平均を 0 に引く）処理を行っておきます。これにより、全体の平均ベクトルは $\mu = 0$ となります。

このデータを、射影したい方向を表す単位ベクトル $u$（制約条件：$u^T u = ||u||^2 = 1$）に投影します。 各データ点 $x_i$ を $u$ に射影した後の1次元の値 $y_i$ は、ベクトルの内積によって次のように表されます。

$$y_i = u^T x_i$$

データの平均が 0 であるため、射影後の値 $y_i$ の平均も当然 0 になります。 したがって、射影されたデータ群の分散 $V(y)$ は、二乗誤差の平均値として次のように展開されます。

$$V(y) = \frac{1}{N} \sum_{i=1}^{N} y_i^2 = \frac{1}{N} \sum_{i=1}^{N} (u^T x_i)^2$$ ここで、$(u^T x_i)^2 = (u^T x_i)(u^T x_i)^T = u^T (x_i x_i^T) u$ と書き換えることができます。 $$V(y) = u^T \left( \frac{1}{N} \sum_{i=1}^{N} x_i x_i^T \right) u$$

ここで、丸括弧で囲まれた中身の式は、データのばらつき度合いを行列として表した「共分散行列（分散共分散行列）$S$」そのものです。

$$S = \frac{1}{N} \sum_{i=1}^{N} x_i x_i^T$$

これを用いると、私たちが最大化したい射影後の分散 $V(y)$ は、驚くほどスッキリとした行列の二次形式にまとめることができます。

$$V(y) = u^T S u$$
Step 04

ラグランジュの未定乗数法と固有値問題への帰着

これで解くべき数学的課題が明確になりました。 「制約条件 $u^T u = 1$ の下で、二次形式 $u^T S u$ を最大化する単位ベクトル $u$ を求める」という最適化問題です。 等式制約付き最適化問題を解くために、数学の強力なツールであるラグランジュの未定乗数法を使用します。

未定乗数 $\lambda$ を導入し、ラグランジュ関数 $L(u, \lambda)$ を以下のように定義します。

$$L(u, \lambda) = u^T S u - \lambda(u^T u - 1)$$

この関数を、求めたいベクトル $u$ で偏微分して 0 とおきます。 ベクトルの微分公式 $\frac{\partial (u^T A u)}{\partial u} = 2Au$（$A$ が対称行列のとき）および $\frac{\partial (u^T u)}{\partial u} = 2u$ を用いて展開します。

$$\frac{\partial L}{\partial u} = 2Su - 2\lambda u = 0$$ $$Su = \lambda u$$

この式は、線形代数で最も重要な式の一つである「固有値方程式」そのものです。 共分散行列 $S$ に対し、ベクトル $u$ に $S$ を作用させても向きが変わらず、長さが $\lambda$ 倍になるような特別な方向を示しています。 この結果は、私たちが求める最適な射影方向 $u$ は、データの共分散行列 $S$ の固有ベクトルに他ならないことを意味します。

さらに、このときの最大化された分散 $V(y)$ の大きさ自体を計算してみます。 $$V(y) = u^T S u = u^T (\lambda u) = \lambda (u^T u)$$ 制約条件である $u^T u = 1$ を適用すると、驚くべき結果が得られます。

$$V(y) = \lambda$$

射影後の分散の大きさは、共分散行列の固有値 $\lambda$ そのものになります。 したがって、分散を最も大きくする方向は、共分散行列の「最大の固有値 $\lambda_1$」に対応する「第1固有ベクトル $u_1$」の向きになります。 これを「第1主成分（PC1）」と呼びます。 次に分散が大きい直交する方向は、2番目に大きな固有値に対応する固有ベクトルとなり、これが「第2主成分（PC2）」となります。

💡 線形代数と統計学の融合：
抽象的で何の役に立つのか分かりにくかった「固有値」や「固有ベクトル」という線形代数の概念が、共分散行列を介することで「データが最も広く分布する主軸の向き（固有ベクトル）」と「その主軸におけるばらつきの量（固有値）」という極めて具体的な意味として結びつきます。

Step 05

実務におけるPCAの応用と「寄与率」

主成分分析によって得られた各主成分が、元データの全情報をどれくらいカバーできているのかを示す指標を「寄与率」と呼びます。 各固有値 $\lambda_j$ を用いて、第 $j$ 主成分の寄与率 $C_j$ は以下のように計算されます。

$$C_j = \frac{\lambda_j}{\sum_{k=1}^{d} \lambda_k}$$

すべての固有値の合計（分母）に対する、着目する主成分の固有値（分子）の割合を意味します。 例えば、第1主成分と第2主成分の寄与率の合計（累積寄与率）が 80% を超えている場合、元のデータが10個の変数を持っていたとしても、第1主成分と第2主成分の2つの合成変数に置き換えるだけで、全体の情報の8割を維持したまま2次元平面に綺麗に可視化することができます。

実務においては、画像の圧縮処理（固有顔を用いた顔認識など）や、アンケート分析における類似する質問項目の集約、大量の財務指標をいくつかの総合スコアにまとめる場面などで、PCAが非常に重宝されています。

Step 06

まとめ

次元削減（PCA）は、情報の損失（分散の低下）を防ぎながらデータを要約するための幾何学的なアプローチです。 分散を最大にする方向を探す等式制約付き最適化問題は、ラグランジュの未定乗数法によって、データの共分散行列の固有値方程式を解く問題へと美しく帰着します。 線形代数の固有値・固有ベクトルの幾何学的意味を腹落ちさせることで、機械学習の次元削減アルゴリズムを数学的な自信を持って使いこなせるようになります。
