Chapter 02 / Unit 15

分数の加法と減法

Step 01

分母が同じ分数の足し算・引き算

分母が同じ分数どうしは、分子だけを足し算・引き算します。分母は変わりません。

例

$\frac{3}{7} + \frac{2}{7} = \frac{3+2}{7} = \frac{5}{7}$

$\frac{5}{8} - \frac{3}{8} = \frac{5-3}{8} = \frac{2}{8} = \frac{1}{4}$（約分も忘れずに）

分母が同じということは、「単位が同じ」ということです。$3$個 $+$ $2$個 $= 5$個 と同じ考え方です。

Step 02

分母が異なる分数の足し算・引き算

分母が異なる場合は、まず通分してから計算します。

例：$\frac{1}{3} + \frac{1}{4}$

  1. 通分: $3$ と $4$ の最小公倍数は $12$
  2. $\frac{1}{3} = \frac{4}{12}$、$\frac{1}{4} = \frac{3}{12}$
  3. 分子を足す: $\frac{4}{12} + \frac{3}{12} = \frac{7}{12}$

$\frac{1}{3} + \frac{1}{4} = \frac{4}{12} + \frac{3}{12} = \frac{7}{12}$

引き算も同じ手順

$\frac{3}{4} - \frac{1}{6} = \frac{9}{12} - \frac{2}{12} = \frac{7}{12}$

Step 03

帯分数の足し算・引き算

帯分数（$1\frac{2}{3}$ など）は、仮分数に直してから計算するのが確実です。

例：$1\frac{2}{3} + 2\frac{1}{4}$

  1. 仮分数に直す: $1\frac{2}{3} = \frac{5}{3}$、$2\frac{1}{4} = \frac{9}{4}$
  2. 通分: $\frac{5}{3} = \frac{20}{12}$、$\frac{9}{4} = \frac{27}{12}$
  3. 足す: $\frac{20}{12} + \frac{27}{12} = \frac{47}{12} = 3\frac{11}{12}$
Step 04

ビジネスでの例：業務時間の計算

例

ある業務に、月曜は $\frac{2}{3}$ 時間、火曜は $\frac{3}{4}$ 時間かかりました。合計時間は？

$\frac{2}{3} + \frac{3}{4} = \frac{8}{12} + \frac{9}{12} = \frac{17}{12} = 1\frac{5}{12}$ 時間

$1$時間と $25$分（$\frac{5}{12}$時間 $= 25$分）という意味になります。

分数の足し算・引き算ができれば、時間や量の計算を正確に行えます。

Step 05

まとめ

  • 分母が同じなら分子だけを足し引きする。
  • 分母が異なる場合は通分してから計算する。
  • 帯分数は仮分数に直してから計算すると間違いにくい。
  • 計算後は必ず約分する習慣をつける。
