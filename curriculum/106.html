Chapter 08-06

ベクトル間の距離（ユークリッド、マンハッタン、マハラノビス）

データ分析において、データ点同士の「近さ（類似度）」を正しく測定することは非常に重要です。日常的に用いる直線距離から、データのばらつきや相関関係を自動で補正してくれるマハラノビス距離まで、段階を追ってわかりやすく解説します。

Step 01

数学における「距離」のルール

普段私たちが口にする「距離」という言葉は、数学においては厳密なルール（公理）で定義されています。 ある空間において、2つの要素 $x, y$ の距離を測る関数 $d(x, y)$ は、以下の3つの条件をすべて満たす必要があります。

  1. マイナスの距離は存在しないこと：距離は常に 0 以上です。また、自分自身との距離だけが 0 になります。
    $d(x, y) \ge 0$ かつ $d(x, y) = 0 \iff x = y$
  2. 行きと帰りの距離は同じであること：$x$ から $y$ への距離と、$y$ から $x$ への距離は等しくなります。
    $d(x, y) = d(y, x)$
  3. 寄り道をすると遠回りになること（三角不等式）：直接向かう距離は、寄り道した合計距離以下になります。
    $d(x, y) \le d(x, z) + d(z, y)$

このルールを満たす計算方法のうち、実務で重要となる3つの距離を紹介します。

Step 02

直線距離（ユークリッド）と街路の距離（マンハッタン）

1つ目は、最も直感的なユークリッド距離（直線距離）です。 2つの点をまっすぐ結んだ直線の長さを指します。 2次元平面上の2点 $x = [x_1, x_2]^T$ と $y = [y_1, y_2]^T$ については、中学校で学ぶ三平方の定理を用いて計算します。

$$d_E(x, y) = \sqrt{(x_1 - y_1)^2 + (x_2 - y_2)^2}$$

これを多次元ベクトルに拡張した数式が以下です。 $$d_E(x, y) = \sqrt{\sum_{i=1}^{n} (x_i - y_i)^2}$$

2つ目は、マンハッタン距離（格子距離）です。 これは碁盤の目のようになっている街路を、道に沿って歩くときの移動距離に相当します。 斜め移動を行わずに、各変数の差の絶対値を単純に足し合わせることで算出します。

$$d_M(x, y) = |x_1 - y_1| + |x_2 - y_2|$$

それぞれの違いは、目標に到達するまでのアプローチにあります。ユークリッド距離は直線距離を測り、マンハッタン距離は直角に曲がるステップ数を累積します。

Step 03

直線距離の弱点：データの「ばらつき」と「相関」

実務のデータを扱う際、ユークリッド距離には大きな弱点があります。 それは、「データのばらつき度合い（分散）」や「変数同士の関連性（相関）」を考慮しない点です。

具体例として、健康診断データを考えてみます。 「身長（メートル単位）」と「体重（キログラム単位）」の2つの変数があるとします。

  • 身長のばらつき：個人差はおよそ 1.5m 〜 1.9m の範囲にとどまり、標準偏差はわずか 0.08m 程度です。
  • 体重のばらつき：個人差はおよそ 40kg 〜 90kg の範囲に広がり、標準偏差は 10kg 程度です。

この2つの変数をそのまま使ってユークリッド距離を計算すると、数値の桁が圧倒的に大きい「体重の差」ばかりが距離に反映されてしまい、「身長の差」が無視されてしまいます。

さらに、身長と体重には強い正の相関（身長が高い人は、体重も重い傾向）があります。 この空間において、平均値から同じユークリッド距離だけ離れた以下の2つの人を比較します。

  • 人A：身長も体重も平均より少し高い（右上方向の分布）
  • 人B：身長は極めて低いが、体重は非常に重い（左上方向の分布）

人Aはよくいる標準的な体型であるのに対し、人Bは確率的に非常に稀な体型です。 しかし、ユークリッド距離の視点では、人Aと人Bは平均値から「全く同じ距離」と判定されてしまいます。 この不公平さを解決するために、データのばらつきを標準化して距離を測定する手法が必要となります。

Step 04

マハラノビス距離を1次元から理解する

マハラノビス距離の数式は難しそうに見えますが、「1次元データの標準化」から考えると非常に簡単です。

1次元のデータ（例えば、テストの点数）において、平均値からの距離を測る場面を考えます。 単なる差の絶対値 $|x - \mu|$ では、テストごとのばらつき（標準偏差 $\sigma$）を考慮できません。 そこで、差を標準偏差 $\sigma$ で割り算することで、ばらつきを揃えた標準化距離を計算します。

$$d = \frac{|x - \mu|}{\sigma}$$

この式を、多次元への拡張を見据えて二乗の形に書き直します。

$$d^2 = (x - \mu) \frac{1}{\sigma^2} (x - \mu) = (x - \mu) (\sigma^2)^{-1} (x - \mu)$$

この1次元の式を、2次元以上の「多次元ベクトル」にそのまま引き延ばしたものがマハラノビス距離の定義式です。 1次元の「分散 $\sigma^2$」を多次元の「共分散行列 $S$」に置き換え、「逆数 $1/\sigma^2$」を「逆行列 $S^{-1}$」に置き換えます。

マハラノビス距離の定義： $$d_{Mah}(x, \mu) = \sqrt{(x - \mu)^T S^{-1} (x - \mu)}$$

この逆行列 $S^{-1}$ を中心に挟み込む計算処理によって、以下の2つの標準化作業が自動で行われます。

  • ばらつき（スケール）の補正：ばらつきの大きい変数軸方向の距離を縮め、ばらつきの小さい方向の距離を拡大して評価します。
  • 相関（関連性）の補正：変数同士の連動パターンに沿った変化を「自然なズレ」と判断し、そこから外れた変化を「不自然な遠いズレ」として鋭く検出します。

結果として、データの分布が右上に向かって細長く伸びている場合、等距離ラインは真円ではなく、データの分布と同じ向きに傾いた「楕円形」になります。

💡 直感的な理解のまとめ：
・ユークリッド距離：定規で測った純粋な長さです。
・マンハッタン距離：道を曲がりながら進む道なり長さです。
・マハラノビス距離：データのばらつきや相関関係を割り算して、確率的な不公平さをなくした「データ分布に照らし合わせた距離」です。

Step 05

まとめ

ユークリッド距離は直線距離を、マンハッタン距離は軸に沿った経路距離を測る単純な手法であり、直感的に計算できますが、データ自体の持つ「ばらつきのスケール」や「変数間の相関」に対応できません。 マハラノビス距離は、共分散行列の逆行列を用いることで、空間全体をデータのばらつきそのもので正規化する強力な数理アプローチです。 データの構造特性に合わせて適切な距離尺度を選択することが、高度な機械学習モデルを構築するための第一歩となります。
