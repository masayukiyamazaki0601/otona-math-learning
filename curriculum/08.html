Chapter 02 / Unit 08

四則演算の基本法則（乗法・除法）

Step 01

乗法（掛け算）の意味

掛け算（乗法）は、同じ数を何回も足す操作を「まとめて」行う計算です。

乗法の基本

$a \times b = c$

  • $a$ を $b$ 回足し合わせたものが $c$
  • 例: $4 \times 3 = 4 + 4 + 4 = 12$（4を3回足す）

乗法の重要な性質

  1. 交換法則: $a \times b = b \times a$（順番を変えても答えは同じ）
  2. 結合法則: $(a \times b) \times c = a \times (b \times c)$
  3. 単位元の存在: $a \times 1 = 1 \times a = a$（1をかけても変わらない）
  4. ゼロの性質: $a \times 0 = 0 \times a = 0$（0をかけると必ず0になる）
  5. 分配法則: $a \times (b + c) = a \times b + a \times c$

分配法則はとくに重要です。$4 \times (3 + 5) = 4 \times 3 + 4 \times 5 = 12 + 20 = 32$ のように、掛け算を分解して計算できます。暗算をするときによく使うテクニックです。

Step 02

除法（割り算）の意味

割り算（除法）は、「全体を均等に分ける」操作です。掛け算の逆の操作です。

除法の基本

$a \div b = c$

  • $a$ を $b$ 等分すると、1つ分は $c$ になる
  • 例: $12 \div 3 = 4$（12個のりんごを3人で分けると、1人4個）

割り算の注意点

  • 交換法則は 成り立たない
  • 0で割ることはできない（数学の絶対ルール）
  • $a \div b = c$ のとき、$c \times b = a$ が成り立つ（掛け算で検算できる）
Step 03

乗法と除法の関係

掛け算と割り算も、足し算と引き算と同じように逆の関係にあります。

乗法と除法の相互関係

  • $a \times b = c$ ならば $c \div b = a$ かつ $c \div a = b$
  • $a \div b = c$ ならば $c \times b = a$
Step 04

ビジネスでの活用例: 単価と数量

ビジネスで頻繁に登場する「単価 $\times$ 数量 = 金額」の関係は、乗除の典型的な応用です。

例：在庫管理

  • 商品A（単価 $1,500$円）が $12$ 個売れた → 売上は $1,500 \times 12 = 18,000$円
  • 総売上 $45,000$円で、単価 $1,500$円の商品を何個売ったか → $45,000 \div 1,500 = 30$個

このように、掛け算と割り算は「全体の量 = 1つあたり $\times$ 個数」という関係を自在に行き来するために使います。

Step 05

九九の暗唱がなぜ重要か

小学校で必死に覚えた九九（くく）は、単なる暗記物ではありません。九九を瞬時に出せるようになると、頭の中の計算スペースに余裕が生まれ、複雑な計算や応用問題に集中できるようになります。

大人になって計算機やExcelが使える環境でも、九九レベルの暗算ができると「ざっくりした見積もり」が格段に速くなります。

Step 06

まとめ

  • 乗法（掛け算）は「同じ数をまとめて足す」操作。
  • 乗法には交換法則・結合法則・分配法則が成り立つ。
  • 除法（割り算）は「均等に分ける」操作で、0で割ることはできない。
  • 乗法と除法は互いに逆の関係。
  • ビジネスでは「単価 $\times$ 数量 = 金額」の形で頻繁に使う。
